## Installation

```sh
cd discord-24points
npm install
```

## Run

```
npm install
npx tsc  # only if you want to re-compile

# production env
npm start
# test env
npm test
```

## Dependencies

- [discord.js](https://discord.js.org/#/): Including documentations about discord.js
